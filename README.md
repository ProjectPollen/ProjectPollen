===========================================================
                PROJECT POLLEN - v1.00.0
                  The Offline Arcade
===========================================================

Thank you for downloading Project Pollen! This is a 
fully self-contained arcade environment.

--- HOW TO START ---
1. Extract the contents of this ZIP file to a folder.
2. Locate the "index.html" file.
3. Right-click "index.html" and select "Open with" 
   (Chrome, Edge, or Firefox recommended).

--- FEATURES ---
- Multi-Tab Support: Run multiple games at once.
- Sandbox Security: No intrusive ads or popup tabs.
- Custom Themes: Create and save your own UI colors.
- Panic Button: Tap the "\" key to quickly hide the site.

--- CONTROLS ---
- [F] : Toggle Fullscreen (while in a game)
- [Backspace] : Return to Menu
- [\] : Panic Redirect (Instant Hide)

--- TROUBLESHOOTING ---
If a game doesn't load, ensure the "Games" and "Icons" 
folders are in the same directory as "index.html".

Enjoy the arcade!
